# Dishes

Endpoints for retrieving Filipino dish data.

## List All Dishes

```
GET /v1/dishes
```

Returns a paginated list of all dishes in the dataset.

### Query Parameters

| Parameter  | Type    | Default | Description             |
| ---------- | ------- | ------- | ----------------------- |
| `page`     | integer | 1       | Page number             |
| `limit`    | integer | 10      | Results per page        |
| `type`     | string  | —       | Filter by dish type     |
| `occasion` | string  | —       | Filter by occasion      |
| `region`   | string  | —       | Filter by origin region |

### Valid Filter Values

**type:** `main_dish`, `soup`, `dessert`, `snack`, `appetizer`, `breakfast`, `rice_dish`, `noodle_dish`, `kakanin`, `bread_pastry`, `side_dish`, `condiment`, `beverage`, `street_food`

**occasion:** `everyday`, `breakfast`, `merienda`, `fiesta`, `christmas`, `media_noche`, `new_year`, `birthday`, `wedding`, `lent`, `party`, `potluck`, `pulutan`, `cold_weather`, `street_food`

### Example Request

```bash
curl "https://filipino-food-api.ryodgie.workers.dev/v1/dishes?type=soup&page=1&limit=5"
```

### Example Response

```json
{
  "data": [...],
  "meta": {
    "total": 8,
    "page": 1,
    "limit": 5,
    "pages": 2
  }
}
```

## Get a Single Dish

```
GET /v1/dishes/:id
```

Returns a single dish by its slug ID.

### Path Parameters

| Parameter | Type   | Description                          |
| --------- | ------ | ------------------------------------ |
| `id`      | string | Dish slug (e.g. `adobo`, `sinigang`) |

### Example Request

```bash
curl https://filipino-food-api.ryodgie.workers.dev/v1/dishes/sinigang
```

### Example Response

```json
{
  "data": {
    "id": "sinigang",
    "name": "Sinigang",
    "description": "A sour tamarind-based soup with pork...",
    "type": ["soup", "main_dish"],
    "main_ingredients": ["pork", "tamarind", "kangkong", "radish", "eggplant"],
    "occasion": ["everyday", "cold_weather"],
    "flavor_profile": ["sour", "savory"],
    "cooking_method": ["boiled", "stewed"],
    "image_url": "https://pub-2e1172f9672f425f9d4a549a832fbd21.r2.dev/dishes/sinigang.jpg",
    "image_credit": "AI Generated (fal.ai Realistic Vision) hosted on Cloudflare R2",
    "tags": ["tamarind", "soup", "comfort food"]
  }
}
```

### Error Response

```json
{
  "error": {
    "message": "Dish not found",
    "code": "NOT_FOUND",
    "status": 404
  }
}
```
