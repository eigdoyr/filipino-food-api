# Introduction

**Filipino Food API** is an open-source REST API providing structured data on Filipino cuisine — dishes, ingredients, regional origins, occasion tags, flavor profiles, and AI-generated food photography.

## What You Can Build

- Filipino food apps and recipe finders
- Cultural education tools
- Restaurant menu systems
- Food discovery experiences
- Regional cuisine explorers

## Base URL

```
https://filipino-food-api.ryodgie.workers.dev
```

## Current Version

All endpoints are versioned under `/v1`.

```
https://filipino-food-api.ryodgie.workers.dev/v1
```

## What's in the Dataset

- **100 Filipino dishes** with structured data
- Regional origins (where applicable)
- Ingredients, occasion tags, flavor profiles, cooking methods
- AI-generated food photography hosted on Cloudflare R2
