# Quickstart

Get up and running with the Filipino Food API in under 2 minutes.

## Make Your First Request

No API key required for basic usage. Just make a GET request to any endpoint.

### Fetch all dishes

```bash
curl https://filipino-food-api.ryodgie.workers.dev/v1/dishes
```

### Fetch a single dish

```bash
curl https://filipino-food-api.ryodgie.workers.dev/v1/dishes/adobo
```

### Search for dishes

```bash
curl https://filipino-food-api.ryodgie.workers.dev/v1/dishes/search?q=pork
```

## Using in JavaScript

```javascript
const response = await fetch(
  "https://filipino-food-api.ryodgie.workers.dev/v1/dishes/adobo",
);
const { data } = await response.json();
console.log(data.name); // "Adobo"
```

## Using in Python

```python
import requests

response = requests.get(
    'https://filipino-food-api.ryodgie.workers.dev/v1/dishes/adobo'
)
dish = response.json()['data']
print(dish['name'])  # "Adobo"
```

## Rate Limits

Anonymous requests are limited to **60 requests per minute** per IP address.

Exceeded limits return a `429` response with a `Retry-After` header.
